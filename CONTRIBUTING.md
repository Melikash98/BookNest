Thank you for considering contributing to this project.

This repository contains the client application for the Hotel Booking System. The project is developed with Android Studio, Java, and Gradle, so contributions should follow the same stack and project structure.

## How to Contribute

### 1. Fork and Clone
- Fork the repository to your GitHub account.
- Clone your fork locally.
- Open the project in Android Studio.
- Let Gradle sync fully before making any changes.

### 2. Create a Branch
Create a separate branch for each task:

- `feat/<short-description>` for new features
- `fix/<short-description>` for bug fixes
- `refactor/<short-description>` for code cleanup
- `chore/<short-description>` for maintenance tasks

Example:
```bash
git checkout -b feat/navigation-improvement
